# ClientPath CRM Lite

ClientPath CRM Lite is a reusable lead conversion system for small service businesses.

It starts with a landing page and enquiry form, then grows into a simple dashboard where the business owner can manage leads, update status, and follow up faster.

## Phase 0

This repository currently contains the initial project baseline:

- Next.js app scaffold
- Tailwind CSS setup
- Clean placeholder homepage
- Basic project folders for `components`, `lib`, `types`, `docs`, and `supabase`
- `.env.example`
- Starter docs for the build

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Project Goal

The MVP is designed to help service businesses:

- capture more enquiries
- avoid losing leads in DMs or email
- follow up faster
- track status from new lead to won or lost

## Stack

- Next.js
- Tailwind CSS
- Supabase Postgres
- Supabase Auth
- Vercel
